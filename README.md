# category-theory

The book of category theory written in Japanese.

## Usage

### Prerequisites

* $\TeX{}$ compiler
* latexmk
* (optional) ChkTeX
* (optional) latexindent

### How to compile locally

1. Edit `main.tex`
1. Run: `latexmk main.tex`

### How to compile on GitHub

1. Tag your commit.
1. Push the tag.
