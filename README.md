# Category Theory (圏論ノート)

The notebook of category theory written in Japanese.

## How to build

### Requirements

#### building tool

* GNU Make

#### TeX to PDF

* lualatex
* latexmk
* (optional) ChkTeX
* (optional) latexindent

#### TeX to HTML

* Pandoc

#### PDF to PNG

* pdftocairo

### How to build

Run `make` in the root directory.
